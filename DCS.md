# DCS

---

## 📋 프로젝트 정보

---

- 프로젝트 이름 : DCS
- 프로젝트 완성도 : 100%
- 사용한 기술 스택 : React.js, TypeScript

## 📋 프로젝트 설명

---

- 기숙사에 택배가 도착할 시 오프라인으로 명부에 택배 정보를 작성하는 것 대신 온라인으로 택배를 관리하는 서비스입니다.

## 📋 프로젝트 기능

---

- 회원가입
- 로그인
- 계정 별 권한 지급 ( 어드민, 택배 기사, 학생 )
- 택배 정보 등록 ( 택배 기사 계정만 가능 )
- 수령 가능한 택배 확인
- 택배 수령 인증
- 지금까지 수령하였던 택배 확인
- 수령 인증 요청을 어드민 페이지로 확인 ( 어드민 계정만 가능 )
